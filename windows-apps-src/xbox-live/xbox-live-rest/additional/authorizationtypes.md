# <a name="authorization-types"></a>承認のタイプ
 
Xbox Live サービスでは、さまざまな種類の承認を使用して、要求とします。 このトピックでは、標準的な承認の種類について説明します。
 
| 種類| 説明| 
| --- | --- | 
| ユーザー | 呼び出し元は、Xbox Live 権限を持つユーザーです。 | 
| タイトル | 呼び出し元は、信頼されたプラットフォーム上のタイトルです。|
