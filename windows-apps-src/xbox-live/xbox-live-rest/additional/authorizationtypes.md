# <a name="authorization-types"></a>承認のタイプ
 
Xbox Live サービスは、要求と承認の種類の数を使用します。 このトピックでは、標準的な認証の種類について説明します。
 
| 種類| 説明| 
| --- | --- | 
| ユーザー | 呼び出し元が、承認された Xbox Live ユーザーです。 | 
| Title (タイトル) | 呼び出し元が、信頼されたプラットフォーム上のタイトルです。|
