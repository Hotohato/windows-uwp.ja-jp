# <a name="mediamarketplaceidcontentrating"></a>/media/{marketplaceId}/contentRating
コンテンツのレーティング トークンにアクセスします。 これらの Uri のドメインが`eds.xboxlive.com`します。
 
  * [URI パラメーター](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>URI パラメーター
 
| パラメーター| 種類| 説明| 
| --- | --- | --- | 
| marketplaceId| string| 必須。 文字列から取得した値、 <b>Windows.Xbox.ApplicationModel.Store.Configuration.MarketplaceId</b>します。| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>有効なメソッド
