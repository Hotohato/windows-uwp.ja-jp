# <a name="getclipresponse-json"></a>GetClipResponse (JSON)
ゲーム クリップをラップします。 
<a id="ID4EN"></a>

 
## <a name="getclipresponse"></a>GetClipResponse
 
GetClipResponse オブジェクトでは、次の仕様があります。
 
| メンバー| 種類| 説明| 
| --- | --- | --- | 
| ゲーム クリップだった| [ゲーム クリップだった](json-gameclip.md)| 問題がなければ、クエリを 1 つのゲーム クリップします。|
