# <a name="xbox-live-storage-platform---connected-storage-title-storage"></a>Xbox Live ストレージ プラットフォーム - 接続ストレージ、タイトル ストレージ

Xbox Live では、発行元がグローバル タイトル データとプレイヤー固有のデータをクラウドに保存できます。

## <a name="in-this-section"></a>このセクションの内容

[接続ストレージ](connected-storage/connected-storage-technical-overview.md)  
接続ストレージ API を使用して格納されたデータは、ユーザー用に PC と複数の Xbox One 本体に自動的にローミングされ、オフラインでも使用できます。

[Xbox Live タイトル ストレージ](xbox-live-title-storage/xbox-live-title-storage.md)  
Xbox Live タイトル ストレージ サービスでは、タイトルのゲーム情報をクラウドに格納できます。 どのプラットフォームで実行されるゲームでも、このサービスを使用できます。
