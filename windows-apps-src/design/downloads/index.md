# <a name="design-toolkits-and-samples-for-uwp-apps"></a>UWP アプリ用の設計ツールキットとサンプル
 

このセクションでは、UWP アプリの設計と UI に関連するダウンロードを紹介します。 Visual Studio など、その他のツールについては、[メインのダウンロード ページ](https://developer.microsoft.com/downloads)をご覧ください。 


## <a name="design-toolkits"></a>設計ツールキット

これらのツールキットは、UWP アプリをデザインするためのコントロールとレイアウトのテンプレートを提供します。

* [Adobe XD ツールキット](https://aka.ms/adobexdtoolkit)
* [Adobe Illustrator ツールキット](https://aka.ms/adobeillustratortoolkit)
* [Adobe Photoshop ツールキット](https://aka.ms/adobephotoshoptoolkit)
* [Framer ツールキット (GitHub)](https://aka.ms/framertoolkit)
* [Sketch ツールキット](https://aka.ms/sketchtoolkit)

## <a name="fonts"></a>フォント

* [Segoe UI と MDL2 アイコン フォント (Mac のみ)](https://aka.ms/SegoeFonts)
* [Hololens のアイコン フォント (Windows のみ)](https://aka.ms/hololensiconfont)
