# Windows アプリの公開

Windows デベロッパー センター ダッシュ ボードを使用すると、アプリを Windows ストアに公開できます。 

> **注** Windows デベロッパー センター ダッシュボードにアクセスして、アプリを申請するには、[開発者アカウント](http://go.microsoft.com/fwlink/p/?LinkId=615100)が必要です。

このセクションには、アプリの公開と管理に関する情報が記載されています。
